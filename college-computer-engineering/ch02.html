### Exception Handling:

try { throw $expression$; } catch (Exception) { $exceptionHandling$; }

catch (Exception) { $exceptionHandling$; }

catch (...) { $handleAllExceptions$; }

Once a catch block receives an exception, it exits the try block and executes next.

In the catch block, you can catch all exceptions of a specific type.

void fun() throw (A, B, C, D)

If not specified, it can throw any type of exception.
